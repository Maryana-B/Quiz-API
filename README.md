# Quiz-API

## Überblick
Es ist ein Quiz-API mit eine Frontend und eine Datenbank.
Ziel: Ein Quiz-System mit Fragen/Antworten.
Andere Eigenschaften, die wir hinzufügen könnten, sind: Ein Timer, zufällige Fragen
### Verwendete Services
	• REST API für Fragen/Antworten, Highscores
	• Datenbank (PostgreSQL)
	• Einfaches Web-Frontend mit HTML/JS
